Using Nixtla Functions

Anomaly

Use this function to detect anomalies in your data. The syntax is:

=NIXTLA_AUTOML_ANOMALY(range, sensibility)

Where:

  • range is a range of cells containing timestamp and value data (two columns).
  • sensibility is a number representing the sensibility of the anomaly detection.

Forecast

Use this function to create a forecast based on your data. The syntax is:

=NIXTLA_AUTOML_FORECAST(range, fh)

Where:

  • range is a range of cells containing timestamp and value data (two columns).
  • fh is a number representing the forecast steps.

Generate Forecast and Anomaly graphs

Forecast

To use Forecast Form, navigate to the "Run Prediction" submenu and click on "Forecast". A sidebar will appear.

Fill in the necessary details:

  • "Range" - input the range of cells containing timestamp and value data.
  • "Forecast steps" - input the number of forecast steps you want.

Click "Submit" to run the forecast.

Anomaly

To use Anomaly Form, navigate to the "Run Prediction" submenu and click on "Anomaly Detection". A sidebar will appear.

Fill in the necessary details:

  • "Range" - input the range of cells containing timestamp and value data.
  • "Sensitivity" - input the sensitivity for the anomaly detection.

Click "Submit" to run the anomaly detection.
